## About

Perception Autonomy Labs is an autonomous systems R&D group focused on retrofit autonomy, vehicle perception, and embedded intelligence.

We work on systems that can bring modern autonomy research and intelligent vehicle capabilities to existing vehicles through modular hardware and software platforms.

Our current research includes:

- Vehicle CAN analysis
- Sensor fusion
- Embedded AI systems
- Real-time telemetry
- GNSS + IMU integration
- Autonomous driving research
- ADAS experimentation
